# SDev (Standard Deviation) PineScript Indicator

## Installation Instructions
1. Copy the code of the SDev indicator.
2. Open TradingView.
3. Go to the Pine Script Editor.
4. Paste the code into the editor.
5. Click on `Save` to save your script.
6. Click on `Add to Chart` to see the indicator in action.

## Features
- Calculates the standard deviation of a chosen dataset.
- Customizable period for calculation.
- Visual representation on the TradingView chart.

## Usage Guide
- Use this indicator to gauge the volatility of a security.
- Typically used in conjunction with other indicators to confirm trends.
- Adjust parameters according to your analysis needs.

## Settings Explanation
- **Length**: Number of periods to calculate the standard deviation.
- **Source**: The data series to use for calculation (e.g., close, open, high, low).
- **Color**: Change the visual color of the indicator for better chart integration.

Feel free to tweak the settings to suit your trading strategy!
